2. Find all the department which have more than and equal 2 Employees.(Use having clause)
Ans:

    -> Where Dept_tb.Dept_id=Emp_tb.Dept_id

    -> Having count(Emp_id)>=2;

5. Write query for to increment sal 10% whose dept_id=1,increament sal 20% whose dept_id=2 and other 30% .(Use only case)


Update Emp_tb set sal = Case Dept_id
    ->
    -> when '1' then sal+(sal*0.1)
    ->
    -> when '2' then sal+(sal*0.2)
    ->
    -> else sal+(sal*0.3)
    -> end
    -> Where Dept_id In('1','2','3')
    -> ;
